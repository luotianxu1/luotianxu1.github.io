| 约束       |             特点             |             |
| :--------- | :--------------------------: | :---------: |
| 主键约束   |          唯一+非空           | primary key |
| 唯一约束   |             唯一             |             |
| 非空约束   |          不能为bull          |             |
| 默认值约束 | 没有指定具体数据时会给默认值 |             |

## 主键约束

### 创建主键

```sql
create table student2
(
 id int primary key,
 name varchar(20)
)
```

### 删除主键

```sql
alter table test2.student2 drop primary key
```
